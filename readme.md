
Introduction
----
- 아이나비 스포츠 m3 제품의 블랙박스 파일을 이어붙이는 스크립트.

Prerequisite
----
```python 3+```

[ffmpeg](https://ffmpeg.org/download.html) 

Usage
----

```mp4concat.py``` 파일을 연 후
```mp4path```는 블랙박스 파일이 저장된 폴더명으로, ```ffmpegpath```는 ffmpeg.exe 파일의 경로로 지정.
그 후 

```
python mp4concat.py
```

이후 results 폴더 확인.
